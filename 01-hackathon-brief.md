# 01 · 比賽簡介

## 基本資訊

**Built with Claude: Life Sciences** — 全球線上黑客松,由 Anthropic、Gladstone Institutes(舊金山生醫研究機構)與 Cerebral Valley 合辦。活動期間 2026 年 7 月 7 日至 13 日,全程線上、不限地區。對象為研究人員、醫師、生物資訊專家與生技開發者。前身系列曾從逾兩萬名申請者中選出 500 名建構者。

> ⚠️ 報名截止為 7 月 5 日,今天(7/7)為活動起始日。

## 兩個賽道

**實驗室組(Laboratory Track)** — 用 Claude Science(Anthropic 新的 AI 科研工作台)探索特定生物學問題,繳交可重現的分析或訓練好的模型。Gladstone 等實驗室提供獨家真實資料集,包括免疫 T 細胞定序、DNA 調控活性預測、蛋白質交互作用網路。

**開發組(Development Track)** — 用 Claude Code 做出能實際給研究人員、診所或生技公司使用的軟體工具。

## 獎勵

前 500 名入選者獲一個月 Claude Max 20x 使用權 + 200 美元 API 額度;得獎隊伍瓜分總額 **10 萬美元** 的 API 與使用額度。

## 評審 = 策略的錨

作品由 **Anthropic + Gladstone** 團隊評審並 demo。這決定了整套取捨(見 `05-competition-strategy.md`)。

## 對本案最關鍵的一步:橋接兩個賽道

我們交的是一個**開發組工具**,但它的引擎裡跑的是**實驗室組等級的推理**,而且——若走 A 路線——吃的是**評審自己提供的真實資料集**。這一橋接是把「酷炫包裝」與「這真能用」切開的關鍵刀口。
